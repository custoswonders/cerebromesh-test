# Pixi.js Candidate Test — Pixel Drift

## Purpose
This is a short take‑home test for a Web Game Developer role. The codebase is scaffolded, and the core gameplay logic is intentionally incomplete.

## Rules
- Do not use AI tools (including ChatGPT, Copilot, or similar) for this test.
- You may use official documentation for Pixi.js and JavaScript.
- Keep your solution clear and readable.

## Your Task
Implement the missing gameplay logic inside `src/game/` (TODOs are already marked):
- Player movement (WASD or Arrow keys).
- Spawn collectibles and obstacles over time.
- Collision detection and responses.
- Score and health updates.
- Game‑over state when health reaches zero.

## Setup
```
cd test_project
npm install
npm run dev
```

## What We Evaluate
- Game loop correctness.
- Collision handling and state management.
- Code clarity and organization.
- Attention to detail (clamping, cleanup, UI).
